Purchase Order Status Approvals

A tool that adds statuses to your sales process, allowing you to manage more efficiently!

Flexible configuration and management

Simple enable or disable options!

As shown in the attached example images, the configuration is very simple, you only need to activate or deactivate the function of each of the states added in the sale so that the system knows if it has to go through one or the other...

Group configuration for each user

You can limit the activity of users by the group they are in.

The Administrator (s) can assign the permissions to the corresponding users through the groups for each state they activate, according to the position they hold.
